CSS Flex vs CSS Grid

artikkeln ble gjort 01.10.2018

De mest populære CSS layoutene er CSS grid og CSS flex. Hovedforskjellen mellom disse er at CSS flex er designet for en dimensjon.

Hvorav CSS grid er designet for to dimensjoner rekker og kolonner på samme tid slik som et rutefelt. CCS flex kan være lettere å bruke hvis du bare skal stile en rekke eller en kolonne, som for eksempel en meny, mens CSS grid kan være lønnsomt å bruke hvis du skal stile en hel nettside med header, nav, article, aside og footer. Med CSS grid kan man få lettere oversikt, spesielt hvis man bruker nettleseren Mozilla fireFox. Fordi der har man et verktøy, slik at man kan se griden på nettsiden. Så hvis du skal kontrollere layouten på en rekke ELLER en kolonne så bruk CSS flex. Men hvis man skal kontrollere layouten på en rekke OG kolonne så bruk CSS grid.

Det finnes også noe som heter CSS float, men den brukes kun til veldig simple layouts.
