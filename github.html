Hva er Git og GitHub?

artikkeln ble gjort 01.10.2018

Git er en versjon kontroll system som har oversikt over alle forandringen i filene dine. GitHub er et program som lagrer alt av sorce code på nettet sånn som en cloud, den gjør det også mulig for flere programmerere å jobbe på samme Project ved hjelp av Git. I motsetning til en vanlig cloud tjeneste kan man jobbe samtidig på samme fil, det vil si at flere kan jobbe på samme fil uten at man overskriver hverandre. Dette gjør GitHub ekstremt gunstig å bruke, hvis man jobber på et stort Project sammen med flere mennesker.
